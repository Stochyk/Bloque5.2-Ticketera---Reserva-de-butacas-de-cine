# Bloque5.2-Ticketera---Reserva-de-butacas-de-cine
Selección de asientos numerados para un show en vivo, previniendo que dos personas reserven la misma butaca simultáneamente


## Funcionamiento

1. El usuario selecciona una butaca.
2. La butaca queda bloqueada durante cinco minutos.
3. Si el pago se confirma, la reserva queda registrada.
4. Si el tiempo expira, la butaca vuelve a estar disponible para otros usuarios.
